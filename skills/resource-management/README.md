# 资源管理 Skills

这个目录包含用于维护社区资源库的 Skills。

## Skills 列表

### resource-search
**资源搜索 Skill** - 搜索最新的 Claude Code Skills 相关资料

- 功能：全网搜索相关资源
- 支持：博客文章、GitHub 仓库、视频教程、案例研究、工具插件
- 输出：格式化的资源列表

### resource-briefing
**资源简报生成 Skill** - 生成资源库的定期简报

- 功能：生成周报、月报或自定义时间段的简报
- 内容：新增资源统计、重点资源推荐、趋势分析
- 输出：格式化的简报文档

### daily-report
**更新日报生成 Skill** - 生成每日更新报告

- 功能：记录当天的资源添加、修改和删除操作
- 内容：更新统计、详细更新列表、待办事项
- 输出：格式化的日报文档

## 使用方法

1. 启用对应的 Skill
2. 使用相应的指令触发功能
3. 将输出结果归档到 `community-resources/` 目录

## 维护流程

1. **搜索** - 使用资源搜索 Skill 搜索最新资料
2. **归档** - 将搜索结果添加到对应的资源文件中
3. **记录** - 使用更新日报生成 Skill 记录更新
4. **总结** - 定期使用资源简报生成 Skill 生成简报
