# 资源模板

使用此模板添加新资源到对应的分类文件中。

## 博客文章模板

```markdown
- **{标题}**
  - 链接：{URL}
  - 描述：{简要描述}
  - 作者：{作者名称}
  - 发布日期：{YYYY-MM-DD}
  - 标签：{标签1}、{标签2}
  - 添加日期：{YYYY-MM-DD}
```

## GitHub 仓库模板

```markdown
- **{仓库名称}**
  - 链接：{GitHub URL}
  - 描述：{简要描述}
  - 作者：{作者/组织}
  - Stars：{数量}
  - 标签：{标签1}、{标签2}
  - 添加日期：{YYYY-MM-DD}
```

## 视频教程模板

```markdown
- **{视频标题}**
  - 链接：{URL}
  - 平台：{YouTube/Bilibili/其他}
  - 时长：{XX:XX}
  - 描述：{简要描述}
  - 作者：{频道/作者}
  - 发布日期：{YYYY-MM-DD}
  - 标签：{标签1}、{标签2}
  - 添加日期：{YYYY-MM-DD}
```

## 案例研究模板

```markdown
- **{项目名称}**

  - 行业：{行业类型}
  - 描述：{简要描述}

  - 标签：{标签1}、{标签2}

```

## 工具和插件模板

```markdown
- **{工具名称}**
```
