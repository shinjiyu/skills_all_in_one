# 贡献指南

欢迎为 Claude Code Skills 教学系列贡献内容！

## 如何贡献

### 报告问题

如果发现任何问题或有改进建议，请创建 Issue。

### 提交代码

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 代码规范

- 遵循现有代码风格
- 添加必要的注释
- 确保代码可运行
- 更新相关文档

### 文档贡献

- 保持文档清晰易懂
- 提供实际示例
- 检查拼写和语法

## 贡献类型

- 修复错误
- 添加新示例
- 改进文档
- 优化代码
- 添加测试

感谢您的贡献！
