## 关于“资源库同步”

本 GitHub Pages 站点由仓库内容自动生成：

- 你在 `community-resources/` 里新增/更新资源后，只要 **commit + push**，GitHub Actions 会自动同步更新 Pages
- 如需本地预览，可运行生成脚本并使用 mkdocs 本地预览（见仓库中的脚本说明）
