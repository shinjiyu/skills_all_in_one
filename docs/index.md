# Skills All In One

本仓库是 **Skills 教学系列** 的示例代码与配套资料库，并持续维护一个可检索的 **Skills 社区资源库**。

## 快速入口

- **教程系列**：见左侧导航「教程系列」
- **社区资源库**：见左侧导航「社区资源库」
- **仓库主页**：`https://github.com/shinjiyu/skills_all_in_one`

## 关于“资源库同步”

本 GitHub Pages 站点由仓库内容自动生成：

- 你在 `community-resources/` 里新增/更新资源后，只要 **commit + push**，GitHub Actions 会自动同步更新 Pages
- 如需本地预览，可运行生成脚本并使用 mkdocs 本地预览（见仓库中的脚本说明）
