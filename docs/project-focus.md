# 项目定位说明

## 项目核心定位

本项目专注于 **Skills 开发和使用**，不是 Claude Code 的使用教程。

## 收录范围

### ✅ 应该收录的内容

1. **Skills 开发教程**
   - 如何编写 SKILL.md 文件
   - Skills 开发最佳实践
   - Skills 架构和设计模式

2. **Skills 示例和案例**
   - 开源 Skills 项目
   - Skills 使用案例
   - Skills 实战经验

3. **Skills 相关工具**
   - Skills 开发工具
   - Skills 测试工具
   - Skills 管理工具

4. **Agent Skills 文档**
   - 官方 Skills 文档
   - Skills API 参考
   - Skills 规范说明

### ❌ 不应该收录的内容

1. **Claude Code 使用教程**
   - Claude Code 安装教程
   - Claude Code 配置教程
   - Claude Code 使用技巧（除非专门讲 Skills）

2. **与 Skills 无关的内容**
   - 一般性的 AI 编程教程
   - Claude Code 功能介绍（非 Skills）
   - 其他 AI 工具的使用教程

## 搜索关键词指南

### 推荐关键词（专注于 Skills）

- `Agent Skills`
- `create Skills`、`build Skills`
- `SKILL.md examples`
- `Skills best practices`
- `Skills GitHub repository`
- `Claude Skills tutorial`

### 避免的关键词（会搜到不相关内容）

- `Claude Code tutorial`（太宽泛）
- `Claude Code install`（安装教程）
- `Claude Code usage`（使用教程）
- `Claude Code features`（功能介绍）

## 资源筛选标准

在添加资源时，请检查：

1. **是否与 Skills 相关？**
   - 是 → 继续检查
   - 否 → 排除

2. **是否专注于 Skills 开发？**
   - 是 → 可以收录
   - 否（主要是 Claude Code 使用）→ 排除

3. **是否包含 SKILL.md 相关内容？**
   - 是 → 优先收录
   - 否 → 谨慎收录
