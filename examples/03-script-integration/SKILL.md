# 脚本集成 Skill

## 描述

这个 Skill 演示了如何将 Python 脚本集成到 Skills 中，用于执行文件处理任务。

## 用途

统计文本文件中的字符数、单词数和行数。

## 指令

当用户要求统计文件信息时，请按照以下步骤操作：

1. **参数说明：**
   - `file_path`: 文件路径（必填）

2. **处理流程：**
   - 检查文件是否存在
   - 调用 `scripts/file_stats.py` 脚本处理文件
   - 传递文件路径作为参数
   - 返回脚本执行结果

3. **输出格式：**
   ```
   文件统计信息：
   - 字符数：XXX
   - 单词数：XXX
   - 行数：XXX
   ```

## 脚本调用

使用以下命令调用脚本：
```bash
python scripts/file_stats.py <file_path>
```

## 示例

**用户输入：** "统计文件 /path/to/file.txt 的信息"

**预期输出：**
```
文件统计信息：
- 字符数：1234
- 单词数：234
- 行数：56
```
